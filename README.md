###  Structural Uncertainty Quantification using KL Expansion (MSc Project)
<a href="https://github.com/Roy16Keane/Structural-analysis-of-a-truss-bridge-under-uncertainties-using-Karhunen-Loeve-Expansion">Repository</a>

- Developed a probabilistic modelling framework for analysing structural uncertainty in a 2D truss bridge system  
- Implemented Karhunen–Loève (KL) expansion for efficient uncertainty propagation  
- Benchmarked KL expansion against Monte Carlo simulation, analytical methods, and neural networks  
- Demonstrated significantly lower computational cost compared to Monte Carlo methods while maintaining accuracy  
- Applied finite element methods (FEM) to model structural deflection under stochastic loading
